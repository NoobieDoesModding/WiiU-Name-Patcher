# WiiU-Name-Patcher
A website that allows the user to make a splatoon username bypassing the 10 character limit. The limit is set as 16. The website is directly built upon vyrval's website originally for JGeckoU or TCPGecko idk. It can make a TCPGecko code for the wiiu and can also make Cemu patches, in .asm and .txt which aren't working currently.
